# DisplayXR Shell

Spatial window manager for tracked 3D displays. Turns a single 3D display into a multi-app spatial desktop where windows float in 3D space with correct stereo rendering.

## Download

Go to [**Releases**](https://github.com/DisplayXR/displayxr-shell-releases/releases) and download the latest installer for your platform:

| Platform | Installer |
|----------|-----------|
| Windows  | `DisplayXR-Shell-Setup.exe` |
| macOS    | `DisplayXR-Shell.dmg` |

## Requirements

- A tracked 3D display supported by DisplayXR
- [DisplayXR Runtime](https://github.com/DisplayXR/displayxr-runtime) installed

## Features

- Spatial windowing — place application windows in 3D space
- Stereo rendering via the DisplayXR runtime
- Multi-app compositing on a single tracked display
- Mouse/keyboard interaction with gaze-based hover
- Layout presets and persistence
